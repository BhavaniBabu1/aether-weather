# 🌤 Aether Weather

A modern and interactive weather web application that provides real-time weather updates, forecasts, and visual insights.

---

## 🚀 Features

* 🌍 Search weather by city name
* 📍 Get weather using your location
* 🎙 Voice search support
* 📊 Temperature & humidity charts (Chart.js)
* 🗺 Interactive map (Leaflet)
* 📅 5-day forecast & hourly forecast
* 🌈 Dynamic background based on weather
* 💡 Smart weather tips
* 🌓 Light/Dark theme toggle
* 🕘 Recent search history

---

## 🛠 Tech Stack

* HTML5
* CSS3 (Glassmorphism UI)
* JavaScript (Vanilla JS)
* OpenWeather API
* Chart.js
* Leaflet.js

2. Open `index.html` in your browser

---

## 🔑 API Setup

This project uses OpenWeather API.

1. Go to https://openweathermap.org/
2. Create an account
3. Generate your API key
4. Replace this line in `script.js`:

```js
const API_KEY = "your_api_key_here";
```
